⏱️ 極簡 168 節食計時器 (Minimal 168 Fasting Timer)

這是一個專為手機瀏覽器設計的「單網頁 (Single HTML File) / 零依賴」168 斷食計時器。
主打極致輕量、無須網路也能完美運作，且絕不會因為關閉瀏覽器而遺失倒數進度。

✨ 核心特色 (Features)

極致輕量：沒有 React、沒有 Vue、沒有 Tailwind，純粹的 HTML + CSS + Vanilla JS。載入時間 < 100ms。

完全離線可用：使用瀏覽器原生的 localStorage 儲存狀態，斷網、關閉分頁、甚至重開機都不會中斷計時。

智慧跨日推算：忘記按開始？支援回溯自訂開始時間（以 30 分鐘為單位），系統會自動判斷是「今天」還是「昨天」開始的。

手動補錄與成就系統：支援手動補登過去的成功紀錄，並具備總天數、連續天數與成就解鎖提示。

隱私第一：所有數據皆保存在您的設備本地端，無後端伺服器，無任何追蹤代碼。
